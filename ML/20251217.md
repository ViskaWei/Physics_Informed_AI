### 15. 下列哪项不是泊松过程特征？

- A. 时间间隔服从指数分布
- B. 不相交区间事件独立
- C. 时间间隔服从正态分布
- D. 事件数服从泊松分布

**答案：C**

这道题的正确答案是 C. 事件发生的时间间隔服从正态分布。
这一项不是泊松过程的特征。泊松过程中的事件间隔服从的是指数分布（对应选项 A），而不是正态分布。
下面用数学公式为您详细推导和解释每一个特征：
1. 为什么选 C（以及为什么 A 是对的）？
我们来推导一下“事件发生的时间间隔”服从什么分布。
假设泊松过程的速率参数为 $\lambda$（表示单位时间内平均发生的事件次数）。
设 $T_1$ 为第 1 个事件发生的时间（即从 0 到第 1 个事件的时间间隔）。
如果 $T_1 > t$，意味着在时间段 $[0, t]$ 内发生的事件数量 $N(t)$ 为 0。
根据泊松分布的定义（见下文第 3 点）：


$$P(N(t) = k) = \frac{(\lambda t)^k e^{-\lambda t}}{k!}$$
当 $k=0$ 时：


$$P(T_1 > t) = P(N(t) = 0) = \frac{(\lambda t)^0 e^{-\lambda t}}{0!} = e^{-\lambda t}$$
那么，$T_1$ 的累积分布函数 (CDF) 为：


$$F_{T_1}(t) = P(T_1 \le t) = 1 - P(T_1 > t) = 1 - e^{-\lambda t}$$
对 $t$ 求导得到概率密度函数 (PDF)：


$$f_{T_1}(t) = \frac{d}{dt}(1 - e^{-\lambda t}) = \lambda e^{-\lambda t}$$
结论：
上式正是指数分布 (Exponential Distribution) 的公式。
所以，选项 A 正确（是特征），而 选项 C 错误（不是特征，正态分布的公式完全不同）。
2. 解释选项 B：独立增量性
特征描述： 事件在不重叠时间区间内相互独立。
数学表达：
对于任意的时间点 $0 \le t_1 < t_2 < t_3 < t_4$，
时间区间 $(t_1, t_2]$ 内发生的事件数 $N(t_2) - N(t_1)$
与时间区间 $(t_3, t_4]$ 内发生的事件数 $N(t_4) - N(t_3)$
是相互独立的随机变量。
这意味着过去发生的事情不会影响未来事件在独立时间段内的发生概率（这也是“无记忆性”的一种体现）。
3. 解释选项 D：计数服从泊松分布
特征描述： 每个时间区间内事件发生的次数服从泊松分布。
数学表达：
这是泊松过程最基本的定义。对于长度为 $\tau$ 的时间区间（例如 $\tau = t_2 - t_1$），该区间内发生 $k$ 次事件的概率由泊松分布给出：

$$P(N(t+\tau) - N(t) = k) = \frac{e^{-\lambda \tau}(\lambda \tau)^k}{k!}, \quad k = 0, 1, 2, \dots$$
其中：
$\lambda$ 是泊松过程的速率（强度）。
$\tau$ 是时间区间的长度。
$\lambda \tau$ 是该区间内期望发生的事件总数。
总结
A (指数分布)：✅ 正确，间隔时间服从 $f(t)=\lambda e^{-\lambda t}$。
B (相互独立)：✅ 正确，这是泊松过程的独立增量假设。
C (正态分布)：❌ 错误，间隔不是正态分布。
D (泊松分布)：✅ 正确，单位时间内的次数服从泊松分布。
因此，选项 C 是这一题的答案。
